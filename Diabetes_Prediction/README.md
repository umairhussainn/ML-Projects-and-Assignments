# 💉 Diabetes Risk Checker

A machine learning web app to predict diabetes risk using XGBoost.

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Structure
Diabetes_Prediction/
├── app.py
├── models/
│   ├── diabetes_xgb.pkl
│   └── scaler.pkl
├── data/
│   └── diabetes.csv
├── diabetes_prediction.ipynb
